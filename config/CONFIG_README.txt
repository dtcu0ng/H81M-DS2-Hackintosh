This come with 2 config. Just rename what you need and delete what you don't need. The default config (config.plist) is for dGPU machine (Nvidia, AMD).
+ If you have dGPU (Nvidia, AMD), just skip this message.
+ If you only have iGPU (Intel Graphics), delete 'config.plist' and rename 'config_igpu.plist' to 'config.plist'

+ Nếu bạn có dGPU (Nvidia, AMD), đơn giản chỉ cần bỏ qua lời nhắn này.
+ Nếu bạn chỉ có iGPU (Intel Graphics), hãy xóa 'config.plist' và đổi tên 'config_igpu.plist' thành 'config.plist'
